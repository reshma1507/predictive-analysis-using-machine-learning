# predictive-analysis-using-machine-learning

Task 2: Predictive Analysis (Machine Learning)

Objective: Build a machine learning model to predict outcomes based on a specific dataset .

Dataset: MNIST / Fare Prediction Dataset.

Tools: Scikit-Learn, Pandas, NumPy.

Key Deliverables: * Developed a Linear Regression model to predict numerical outcomes.

Demonstrated the full ML pipeline: Feature Selection, Data Splitting (train_test_split), Model Training, and Evaluation.

Evaluated performance using Mean Absolute Error (MAE) and R² Score.
